В этом разделе приведены примеры реализации функциональных фич проекта с использованием **TypeScript**, **React** и *
*TanStack Query**.  
Цель — показать правильную организацию модулей, слоёв и взаимодействие между ними.

- **TypeScript** — строгая типизация для всех слоёв приложения.
- **React** — реализация UI и компонентов.
- **TanStack Query** — управление асинхронными запросами, кэшированием и синхронизацией данных с сервером.

> TanStack Query используется только в слое `app` для получения/мутации данных через порты `domain`. Бизнес-логика
> остаётся независимой от UI и API.
